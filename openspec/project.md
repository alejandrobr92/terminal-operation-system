# TOS Microfrontend Platform

## Overview

A frontend platform simulating a Terminal Operation System (TOS) using microfrontends.

## Goals

- Demonstrate clear domain separation
- Enable independent deployment of MFEs
- Use typed contracts and explicit communication

## Architecture

- Shell (host application)
- Yard Operations MFE
- Move Planning MFE
- Analytics MFE

## Tech Stack

- React + TypeScript + Vite
- Module Federation
- Material UI
- Event Bus for communication

## Constraints

- No Next.js
- Frontend-only (mock data allowed)

## Evaluation Focus

- Architecture clarity
- Contracts
- Communication strategy
- Maintainability
